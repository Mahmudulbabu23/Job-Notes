### বিভিন্ন সংখ্যা পদ্ধতি

| পদ্ধতি | ভিত্তি | অঙ্ক | ব্যবহার |
|---|---|---|---|
| Binary | 2 | 0, 1 | কম্পিউটার |
| Octal | 8 | 0–7 | — |
| Decimal | 10 | 0–9 | দৈনন্দিন |
| Hexadecimal | 16 | 0–9, A–F | রঙ, মেমোরি |

### রূপান্তর পদ্ধতি

**Decimal → Binary:**
```
দশমিককে ২ দিয়ে ভাগ করো, ভাগশেষ নিচ থেকে উপরে লেখো
25 ÷ 2 = 12 R 1
12 ÷ 2 = 6  R 0
6  ÷ 2 = 3  R 0
3  ÷ 2 = 1  R 1
1  ÷ 2 = 0  R 1
∴ 25 (দশমিক) = 11001 (বাইনারি)
```

**Binary → Decimal:**
```
11001 = 1×2⁴ + 1×2³ + 0×2² + 0×2¹ + 1×2⁰
      = 16 + 8 + 0 + 0 + 1 = 25
```

| Spyware | গোপনে তথ্য চুরি করে |
| Adware | বিজ্ঞাপন দেখায় |

| SQL Injection | Database-এ ক্ষতিকর SQL কোড |
| Man-in-the-Middle | দুই পক্ষের মধ্যে তথ্য আটকানো |
| Zero-Day Attack | অজানা দুর্বলতা কাজে লাগানো |
| Brute Force | সব সম্ভাব্য পাসওয়ার্ড চেষ্টা |

```
Q-এর বামে R → RQ
```

```
n ঘণ্টা m মিনিটে কোণ:
θ = |30n - 5.5m|°

উদাহরণ:
৪টা ৪০ মিনিটে কোণ:
= |30×4 - 5.5×40|
= |120 - 220|
= |-100|
= 100°

একদিনে মিনিটের কাঁটা ঘণ্টার কাঁটাকে ধরে ফেলে:
= 65(5/11) মিনিট অন্তর
= মোট ২২ বার মিলিত হয় (২৪ ঘণ্টায়)
```

| NPA | Non-Performing Assets; খেলাপি ঋণ |
| NPL | Non-Performing Loans |
| SWIFT | Society for Worldwide Interbank Financial Telecommunication |
| RTGS | Real Time Gross Settlement |
| EFT | Electronic Fund Transfer |
| ATM | Automated Teller Machine |
| POS | Point of Sale |
| EMV | Europay Mastercard Visa (chip card) |
| IBAN | International Bank Account Number |
| MICR | Magnetic Ink Character Recognition (চেকে) |
| KYC | Know Your Customer |
| AML | Anti-Money Laundering |
| CFT | Combating Financing of Terrorism |
| BASEL | আন্তর্জাতিক ব্যাংক নিয়মকানুন; বর্তমান BASEL III |
