## ৩.১ শতকরা (Percentage) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   শতকরা = (অংশ ÷ সম্পূর্ণ) × ১০০

সূত্র ২:   সংখ্যার x% = সংখ্যা × (x ÷ ১০০)

সূত্র ৩:   x% বৃদ্ধি → নতুন মান = পুরনো × (১ + x/১০০)
            x% হ্রাস → নতুন মান = পুরনো × (১ − x/১০০)

সূত্র ৪:   যদি A, B অপেক্ষা x% বেশি হয়:
            B = A × 100 / (100 + x)

সূত্র ৫:   যদি A, B অপেক্ষা x% কম হয়:
            B = A × 100 / (100 − x)

সূত্র ৬:   ক্রমিক % পরিবর্তন:
            সামগ্রিক % = a + b + (ab/100)
            (a ও b উভয় বৃদ্ধি হলে + , হ্রাস হলে −)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### শর্টকাট ভগ্নাংশ রূপান্তর

| শতকরা | ভগ্নাংশ | শতকরা | ভগ্নাংশ |
|---|---|---|---|
| 10% | 1/10 | 11.11% | 1/9 |
| 12.5% | 1/8 | 16.67% | 1/6 |
| 20% | 1/5 | 25% | 1/4 |
| 33.33% | 1/3 | 50% | 1/2 |
| 66.67% | 2/3 | 75% | 3/4 |
| 40% | 2/5 | 60% | 3/5 |
| 80% | 4/5 | 37.5% | 3/8 |

> 🔥 **Trick:** এই রূপান্তরগুলো মুখে মুখে জানলে গণনা দ্রুত হয়।

### MCQ উদাহরণ

**প্রশ্ন ১:** একটি জিনিসের দাম ২০% বাড়ার পর আবার ২০% কমলে পরিবর্তন কত?

**সমাধান:**
```
সামগ্রিক % পরিবর্তন = 20 + (−20) + (20 × (−20)/100)
                     = 0 − 4 = −4%
∴ দাম 4% কমবে।
```
> উত্তর: **৪% হ্রাস পাবে**

**প্রশ্ন ২:** ৪৮০ টাকার ৩৭.৫% কত?

```
শর্টকাট: ৩৭.৫% = ৩/৮
∴ ৪৮০ × ৩/৮ = ১৮০
```
> উত্তর: **১৮০ টাকা**

**প্রশ্ন ৩:** একটি সংখ্যার ৩০% অপর সংখ্যার ৪০% এর সমান। প্রথম সংখ্যা দ্বিতীয়ের কত%?

```
ধরি, প্রথম = x, দ্বিতীয় = y
x × 30/100 = y × 40/100
x/y = 40/30 = 4/3
x = (4/3) × y
∴ x, y-এর = (4/3) × 100% = 133.33%
```
> উত্তর: **133⅓%**

## ৩.২ লাভ-ক্ষতি (Profit & Loss) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   লাভ (P) = বিক্রয়মূল্য (SP) − ক্রয়মূল্য (CP)
            ক্ষতি (L) = CP − SP

সূত্র ২:   লাভ% = (P / CP) × 100
            ক্ষতি% = (L / CP) × 100

সূত্র ৩:   SP = CP × (100 + লাভ%) / 100
            SP = CP × (100 − ক্ষতি%) / 100

সূত্র ৪:   CP = SP × 100 / (100 + লাভ%)
            CP = SP × 100 / (100 − ক্ষতি%)

সূত্র ৫:   ছাড় (Discount) = তালিকামূল্য (MP) − বিক্রয়মূল্য (SP)
            ছাড়% = (ছাড় / MP) × 100
            SP = MP × (100 − ছাড়%) / 100

সূত্র ৬:   একই দামে দুটি জিনিস কিনে একটিতে x% লাভ ও
            অন্যটিতে x% ক্ষতি হলে:
            সামগ্রিক ক্ষতি% = x² / 100
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** একটি দোকানদার ৩০% লাভে জিনিস বিক্রি করেন। যদি তিনি ১৩ টাকা কম নিতেন তাহলে ১৭% লাভ হত। জিনিসটির ক্রয়মূল্য কত?

```
ধরি CP = x
SP₁ = x × 130/100 = 1.3x
SP₂ = x × 117/100 = 1.17x

পার্থক্য = 1.3x − 1.17x = 0.13x = 13
∴ x = 13/0.13 = 100 টাকা
```
> উত্তর: **১০০ টাকা**

**প্রশ্ন ২:** একটি বই ২০% ছাড়ে ৩৬০ টাকায় বিক্রি হলে তালিকামূল্য কত?

```
SP = MP × (100 − 20)/100
360 = MP × 80/100
MP = 360 × 100/80 = 450 টাকা
```
> উত্তর: **৪৫০ টাকা**

**প্রশ্ন ৩:** একই দামে কেনা দুটো জিনিসে যথাক্রমে ৮% লাভ ও ৮% ক্ষতি হলে মোট লাভ বা ক্ষতি কত?

```
সূত্র: ক্ষতি% = x²/100 = 8²/100 = 64/100 = 0.64%
∴ মোট 0.64% ক্ষতি হবে।
```
> উত্তর: **০.৬৪% ক্ষতি**

## ৩.৩ ছাড় (Discount) — সূত্র ও উদাহরণ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   SP = MP − Discount
            SP = MP × (1 − d/100)

সূত্র ২:   ক্রমিক ছাড় d₁% ও d₂% হলে:
            কার্যকর ছাড়% = d₁ + d₂ − (d₁ × d₂/100)

সূত্র ৩:   ছাড় থাকলেও লাভ করতে হলে:
            SP ≥ CP হতে হবে
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**প্রশ্ন:** ৩০% ও ২০% ক্রমিক ছাড়ের কার্যকর ছাড় কত?

```
কার্যকর ছাড়% = 30 + 20 − (30 × 20/100)
              = 50 − 6 = 44%
```
> উত্তর: **৪৪%** (৫০% নয়!)

## ৩.৪ অনুপাত ও সমানুপাত (Ratio & Proportion)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   a:b অনুপাতে x টাকা ভাগ:
            প্রথম পায় = x × a/(a+b)
            দ্বিতীয় পায় = x × b/(a+b)

সূত্র ২:   a:b = c:d হলে (proportion):
            ad = bc  [ক্রস গুণ]

সূত্র ৩:   Componendo: (a+b)/(a−b) = (c+d)/(c−d)

সূত্র ৪:   যদি a:b = 2:3 এবং b:c = 4:5 হয় তাহলে
            a:b:c = ?
            → a:b = 8:12, b:c = 12:15
            → a:b:c = 8:12:15

সূত্র ৫:   তৃতীয় আনুপাতিক (x) of a, b:
            a/b = b/x → x = b²/a

সূত্র ৬:   চতুর্থ আনুপাতিক (x) of a, b, c:
            a/b = c/x → x = bc/a

সূত্র ৭:   মধ্য আনুপাতিক (x) of a, b:
            a/x = x/b → x = √(ab)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** a:b = 3:4 এবং b:c = 5:6 হলে a:b:c = ?

```
a:b = 3:4 = 15:20
b:c = 5:6 = 20:24
∴ a:b:c = 15:20:24
```

**প্রশ্ন ২:** ৪, ৮ এর তৃতীয় আনুপাতিক কত?

```
তৃতীয় আনুপাতিক = b²/a = 8²/4 = 64/4 = 16
```

**প্রশ্ন ৩:** ৬, ১২ এর মধ্য আনুপাতিক কত?

```
মধ্য আনুপাতিক = √(ab) = √(6×12) = √72 = 6√2
```

## ৩.৫ অংশীদারিত্ব (Partnership)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   মুনাফা বিভাজন = মূলধন × সময়
            A:B:C = (P_A × T_A):(P_B × T_B):(P_C × T_C)

সূত্র ২:   একই সময়ে বিনিয়োগ হলে:
            মুনাফা বিভাজন = মূলধনের অনুপাত

সূত্র ৩:   ঘুমন্ত অংশীদার (Sleeping partner) শুধু
            মূলধন দেয়, কাজ করে না।
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**প্রশ্ন:** A ১০,০০০ টাকা ৮ মাস ও B ১২,০০০ টাকা ৬ মাস বিনিয়োগ করে। মুনাফা ১৪,০০০ টাকা হলে B কত পাবে?

```
A এর ভাগ = 10,000 × 8 = 80,000
B এর ভাগ = 12,000 × 6 = 72,000
অনুপাত = 80,000:72,000 = 10:9

B পাবে = 14,000 × 9/19 = 6,631.57 ≈ 6,632 টাকা
```

## ৩.৬ গড় (Average) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   গড় = মোট রাশির সমষ্টি ÷ রাশির সংখ্যা

সূত্র ২:   মোট সমষ্টি = গড় × রাশির সংখ্যা

সূত্র ৩:   নতুন রাশি যোগ/বাদ দিলে:
            নতুন সমষ্টি = পুরনো সমষ্টি ± নতুন রাশি

সূত্র ৪:   ভুল রাশির কারণে গড়ের পরিবর্তন:
            সঠিক সমষ্টি = ভুল সমষ্টি − ভুল মান + সঠিক মান

সূত্র ৫:   ক্রমিক সংখ্যার গড়:
            গড় = (প্রথম + শেষ) / 2

সূত্র ৬:   n টি ক্রমিক স্বাভাবিক সংখ্যার গড় = (n+1)/2

সূত্র ৭:   ভারযুক্ত গড় (Weighted Average):
            গড় = (w₁x₁ + w₂x₂ + ... + wₙxₙ) / (w₁ + w₂ + ... + wₙ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** ৭টি সংখ্যার গড় ৪৫। তিনটি সংখ্যার গড় ৪০ হলে বাকি চারটির গড় কত?

```
মোট = 7 × 45 = 315
তিনটির মোট = 3 × 40 = 120
বাকি চারটির মোট = 315 − 120 = 195
বাকি চারটির গড় = 195 / 4 = 48.75
```

**প্রশ্ন ২:** ৫টি সংখ্যার গড় ২৪। একটি সংখ্যা ২৬ এর বদলে ভুলে ৬২ লেখা হয়েছে। সঠিক গড় কত?

```
ভুল সমষ্টি = 5 × 24 = 120
সঠিক সমষ্টি = 120 − 62 + 26 = 84
সঠিক গড় = 84 / 5 = 16.8
```

**প্রশ্ন ৩:** ১ থেকে ৯৯ পর্যন্ত বিজোড় সংখ্যার গড় কত?

```
বিজোড় সংখ্যা: 1, 3, 5, ..., 99
সংখ্যার পরিমাণ = 50টি
গড় = (প্রথম + শেষ)/2 = (1 + 99)/2 = 50
```

## ৩.৭ সরল সুদ (Simple Interest) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   SI = PRT / 100
            P = মূলধন, R = বার্ষিক হার%, T = সময় (বছর)

সূত্র ২:   মূলধন P = (SI × 100) / (R × T)

সূত্র ৩:   হার R = (SI × 100) / (P × T)

সূত্র ৪:   সময় T = (SI × 100) / (P × R)

সূত্র ৫:   সুদ-আসল (A) = P + SI = P(1 + RT/100)

সূত্র ৬:   n বছরে সুদ আসলের k গুণ হলে:
            R = 100(k−1) / n

সূত্র ৭:   দ্বিগুণ হতে সময়:
            T = 100 / R  (সরল সুদে)
            Rule of 72: T ≈ 72/R  (চক্রবৃদ্ধিতে)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** কত বছরে ১৫% সরল সুদে একটি সংখ্যা তিনগুণ হবে?

```
তিনগুণ মানে: A = 3P → SI = 2P
SI = PRT/100
2P = P × 15 × T / 100
T = 200/15 = 13.33 বছর ≈ 13⅓ বছর
```

**প্রশ্ন ২:** কত% সরল সুদে ৫ বছরে সুদ আসলের ⅖ অংশ হবে?

```
SI = (2/5)P
SI = PRT/100
(2/5)P = P × R × 5/100
R = (2/5) × 100/5 = 8%
```

**প্রশ্ন ৩:** ৮% সুদে ৫,০০০ টাকার ৩ বছরের সুদ কত?

```
SI = 5000 × 8 × 3 / 100 = 1,200 টাকা
```

## ৩.৮ চক্রবৃদ্ধি সুদ (Compound Interest) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   বার্ষিক চক্রবৃদ্ধি:
            A = P(1 + R/100)ⁿ
            CI = A − P

সূত্র ২:   অর্ধবার্ষিক চক্রবৃদ্ধি:
            A = P(1 + R/200)^(2n)

সূত্র ৩:   ত্রৈমাসিক চক্রবৃদ্ধি:
            A = P(1 + R/400)^(4n)

সূত্র ৪:   ২ বছরে CI ও SI-এর পার্থক্য:
            CI − SI = P(R/100)²

সূত্র ৫:   ৩ বছরে CI ও SI-এর পার্থক্য:
            CI − SI = P(R/100)²(3 + R/100)

সূত্র ৬:   ভিন্ন হারে চক্রবৃদ্ধি:
            A = P(1+r₁/100)(1+r₂/100)(1+r₃/100)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** ৫,০০০ টাকা ১০% বার্ষিক চক্রবৃদ্ধি সুদে ৩ বছরে কত হবে?

```
A = 5000 × (1 + 10/100)³
  = 5000 × (1.1)³
  = 5000 × 1.331
  = 6,655 টাকা
CI = 6655 − 5000 = 1,655 টাকা
```

**প্রশ্ন ২:** ১০% সুদে ২ বছরে ১০,০০০ টাকার CI ও SI-এর পার্থক্য কত?

```
পার্থক্য = P(R/100)² = 10,000 × (10/100)²
          = 10,000 × 0.01 = 100 টাকা
```

**প্রশ্ন ৩:** ৮% সুদে অর্ধবার্ষিক চক্রবৃদ্ধিতে ১ বছরে ১০,০০০ টাকার সুদ কত?

```
A = 10,000 × (1 + 8/200)²
  = 10,000 × (1.04)²
  = 10,000 × 1.0816
  = 10,816 টাকা
CI = 816 টাকা
```

## ৩.৯ সময় ও কাজ (Time & Work) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   A, n দিনে কাজ করলে:
            A-এর এক দিনের কাজ = 1/n

সূত্র ২:   A ও B একসাথে কাজ করলে:
            একদিনে করে = 1/a + 1/b
            একসাথে শেষ করতে সময় = ab/(a+b)

সূত্র ৩:   A, B, C একসাথে:
            সময় = abc/(ab+bc+ca)

সূত্র ৪:   A, B অপেক্ষা k গুণ দক্ষ হলে:
            A, B-এর কাজ 1/k সময়ে করে

সূত্র ৫:   কাজ, পুরুষ ও দিনের সম্পর্ক:
            M₁D₁ = M₂D₂  (একই কাজ)
            M₁D₁H₁ = M₂D₂H₂  (একই কাজ, ঘণ্টাও ভিন্ন)

সূত্র ৬:   কাজের পরিমাণ ভিন্ন হলে:
            M₁D₁/W₁ = M₂D₂/W₂

সূত্র ৭:   A একা x দিনে শেষ করে এবং A ও B একসাথে
            y দিনে শেষ করলে:
            B একা করতে পারে = xy/(x−y) দিনে
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** A ১২ দিনে এবং B ১৮ দিনে একটি কাজ করে। A ৩ দিন কাজ করে চলে গেলে B বাকি কাজ কত দিনে শেষ করবে?

```
A-এর ৩ দিনের কাজ = 3 × 1/12 = 1/4
বাকি কাজ = 1 − 1/4 = 3/4
B একা 3/4 কাজ করতে সময় = (3/4) × 18 = 13.5 দিন
```

**প্রশ্ন ২:** ১৫ জন পুরুষ ২০ দিনে একটি কাজ করতে পারে। ১০ দিনে কাজটি শেষ করতে কতজন লাগবে?

```
M₁D₁ = M₂D₂
15 × 20 = M₂ × 10
M₂ = 300/10 = 30 জন
```

**প্রশ্ন ৩:** A, B অপেক্ষা ৩ গুণ দক্ষ। B একা ৩৬ দিনে কাজটি করতে পারে। A ও B একসাথে কত দিনে করবে?

```
A, B-এর ৩ গুণ দক্ষ → A করে ৩৬/৩ = ১২ দিনে
একসাথে = ab/(a+b) = 12×36/(12+36) = 432/48 = 9 দিন
```

## ৩.১০ পাইপ ও চৌবাচ্চা (Pipes & Cisterns)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   Pipe A, x ঘণ্টায় চৌবাচ্চা পূর্ণ করে:
            A-এর ১ ঘণ্টার কাজ = +1/x (পূর্ণ করে)

সূত্র ২:   Pipe B, y ঘণ্টায় চৌবাচ্চা খালি করে:
            B-এর ১ ঘণ্টার কাজ = −1/y (খালি করে)

সূত্র ৩:   A ও B একসাথে খোলা থাকলে:
            নেট কাজ/ঘণ্টা = 1/x − 1/y
            পূর্ণ হতে সময় = xy/(y−x)  [y > x হলে]

সূত্র ৪:   Leakage (ফুটো) সমস্যা:
            নেট হার = পূর্ণ করার হার − ফুটোর হার
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** একটি পাইপ ৬ ঘণ্টায় চৌবাচ্চা পূর্ণ করে। একটি ফুটো দিয়ে পানি বের হয় এবং পূর্ণ হতে ৮ ঘণ্টা লাগে। ফুটো দিয়ে পানি বের হয়ে পূর্ণ চৌবাচ্চা খালি হতে কত সময় লাগবে?

```
পূর্ণ করার হার = 1/6
ফুটোসহ নেট হার = 1/8
ফুটোর হার = 1/6 − 1/8 = 4/24 − 3/24 = 1/24
∴ ফুটো দিয়ে খালি হতে ২৪ ঘণ্টা।
```

**প্রশ্ন ২:** পাইপ A ১০ ঘণ্টায় পূর্ণ করে, B ১৫ ঘণ্টায় খালি করে। উভয় খোলা থাকলে কত ঘণ্টায় পূর্ণ হবে?

```
নেট হার = 1/10 − 1/15 = 3/30 − 2/30 = 1/30
সময় = 30 ঘণ্টা
```

## ৩.১১ সময়, গতি ও দূরত্ব (Time, Speed & Distance)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   দূরত্ব = গতি × সময়
            গতি = দূরত্ব ÷ সময়
            সময় = দূরত্ব ÷ গতি

সূত্র ২:   একই দূরত্বে ভিন্ন গতিতে গড় গতি:
            গড় গতি = 2ab/(a+b)   [যেখানে a, b দুটো গতি]

সূত্র ৩:   ভিন্ন দূরত্বে ভিন্ন গতিতে গড় গতি:
            গড় গতি = মোট দূরত্ব ÷ মোট সময়

সূত্র ৪:   গতির রূপান্তর:
            km/h → m/s: × (5/18)
            m/s → km/h: × (18/5)

সূত্র ৫:   একই সময়ে বিপরীত দিকে চললে মিলন সময়:
            সময় = দূরত্ব ÷ (গতি₁ + গতি₂)

সূত্র ৬:   একই দিকে চললে ধরার সময়:
            সময় = দূরত্ব ÷ (গতি₁ − গতি₂)

সূত্র ৭:   ট্রেনের সূত্র:
            স্তম্ভ পার হতে সময় = ট্রেনের দৈর্ঘ্য ÷ গতি
            প্ল্যাটফর্ম পার হতে সময় = (ট্রেন + প্ল্যাটফর্ম) ÷ গতি
            বিপরীত ট্রেন পার হতে = (T₁ + T₂) ÷ (S₁ + S₂)
            একই দিকের ট্রেন পার হতে = (T₁ + T₂) ÷ (S₁ − S₂)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** একজন ব্যক্তি ৩০ km/h বেগে যায় ও ২০ km/h বেগে ফেরে। গড় গতি কত?

```
গড় গতি = 2 × 30 × 20 / (30 + 20)
        = 1200/50 = 24 km/h
```

> ⚠️ **সতর্কতা:** (30+20)/2 = 25 — এটা ভুল। দূরত্ব একই হলে 2ab/(a+b) ব্যবহার করতে হবে।

**প্রশ্ন ২:** ৩০০ মিটার দীর্ঘ একটি ট্রেন ৯০ km/h বেগে ৫০০ মিটার দীর্ঘ প্ল্যাটফর্ম পার হতে কত সময় লাগবে?

```
গতি = 90 × 5/18 = 25 m/s
মোট দূরত্ব = 300 + 500 = 800 m
সময় = 800 / 25 = 32 সেকেন্ড
```

**প্রশ্ন ৩:** দুটি শহরের মধ্যে দূরত্ব ৩৬০ km। দুটি ট্রেন যথাক্রমে ৭০ ও ৮০ km/h বেগে পরস্পরের দিকে রওনা হয়। কত ঘণ্টা পর মিলিত হবে?

```
সম্মিলিত গতি = 70 + 80 = 150 km/h
মিলিত হওয়ার সময় = 360 / 150 = 2.4 ঘণ্টা
```

## ৩.১২ নৌকা ও স্রোত (Boat & Stream)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ধরি: নৌকার গতি = b (স্থির পানিতে)
     স্রোতের গতি = s

সূত্র ১:   অনুকূল গতি (Downstream) = b + s
            প্রতিকূল গতি (Upstream) = b − s

সূত্র ২:   নৌকার গতি b = (D + U) / 2
            স্রোতের গতি s = (D − U) / 2
            [D = Downstream, U = Upstream গতি]

সূত্র ৩:   d দূরত্ব অনুকূলে সময় t₁, প্রতিকূলে t₂ হলে:
            নৌকার গতি = d(t₁+t₂) / (2t₁t₂)
            স্রোতের গতি = d(t₂−t₁) / (2t₁t₂)

সূত্র ৪:   স্থির পানিতে কেউ x km/h সাঁতরায়।
            নদীতে আড়াআড়ি পার হলে:
            প্রকৃত গতি = √(x² − s²)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** একটি নৌকা অনুকূলে ৩ ঘণ্টায় ৩৬ km এবং প্রতিকূলে ৪ ঘণ্টায় ২৪ km যায়। স্থির পানিতে নৌকার গতি ও স্রোতের গতি কত?

```
D = 36/3 = 12 km/h
U = 24/4 = 6 km/h

নৌকার গতি = (12 + 6)/2 = 9 km/h
স্রোতের গতি = (12 − 6)/2 = 3 km/h
```

**প্রশ্ন ২:** একটি নৌকার স্থির পানিতে গতি ১৫ km/h। স্রোতের গতি ৩ km/h। ৭২ km অনুকূলে ও প্রতিকূলে যেতে মোট কত সময় লাগবে?

```
অনুকূল গতি = 15 + 3 = 18 km/h
প্রতিকূল গতি = 15 − 3 = 12 km/h

অনুকূলে সময় = 72/18 = 4 ঘণ্টা
প্রতিকূলে সময় = 72/12 = 6 ঘণ্টা
মোট সময় = 10 ঘণ্টা
```

## ৩.১৩ বয়স সমস্যা (Ages)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   n বছর আগের বয়স = বর্তমান বয়স − n
            n বছর পরের বয়স = বর্তমান বয়স + n

সূত্র ২:   বয়সের অনুপাত t বছর আগে a:b ছিল,
            এখন c:d হলে বর্তমান বয়স বের করা যায়।

সূত্র ৩:   A-এর বয়স B-এর k গুণ হলে:
            A-এর বয়স = k × B-এর বয়স
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** পিতার বয়স পুত্রের বয়সের ৪ গুণ। ৫ বছর পরে পিতার বয়স পুত্রের ৩ গুণ হবে। বর্তমান বয়স কত?

```
ধরি পুত্রের বয়স = x, পিতার = 4x
৫ বছর পর: পিতা = 4x+5, পুত্র = x+5
শর্ত: 4x + 5 = 3(x + 5)
4x + 5 = 3x + 15
x = 10

পুত্রের বয়স = ১০ বছর, পিতার = ৪০ বছর
```

**প্রশ্ন ২:** A ও B-এর বয়সের অনুপাত ৫:৩। ৬ বছর পর অনুপাত ৭:৫ হবে। বর্তমান বয়স কত?

```
ধরি A = 5k, B = 3k
৬ বছর পর: (5k+6)/(3k+6) = 7/5
5(5k+6) = 7(3k+6)
25k + 30 = 21k + 42
4k = 12 → k = 3

A = 15 বছর, B = 9 বছর
```

## ৩.১৪ গুরুত্বপূর্ণ বীজগাণিতিক সূত্র — সম্পূর্ণ তালিকা

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
বর্গ সম্পর্কিত:
  (a + b)² = a² + 2ab + b²
  (a − b)² = a² − 2ab + b²
  (a + b)² − (a − b)² = 4ab
  (a + b)² + (a − b)² = 2(a² + b²)
  (a + b)(a − b) = a² − b²

ঘন সম্পর্কিত:
  (a + b)³ = a³ + 3a²b + 3ab² + b³
  (a − b)³ = a³ − 3a²b + 3ab² − b³
  a³ + b³ = (a + b)(a² − ab + b²)
  a³ − b³ = (a − b)(a² + ab + b²)
  a³ + b³ + c³ − 3abc = (a+b+c)(a²+b²+c²−ab−bc−ca)
  যদি a + b + c = 0 হয়: a³ + b³ + c³ = 3abc

চতুর্থ শক্তি:
  a⁴ − b⁴ = (a²+b²)(a+b)(a−b)

গুরুত্বপূর্ণ পরিচয় সূত্র:
  a² + b² = (a+b)² − 2ab
  a² + b² = (a−b)² + 2ab
  (a+b+c)² = a²+b²+c²+2ab+2bc+2ca
  a² + b² + c² = (a+b+c)² − 2(ab+bc+ca)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**প্রশ্ন ১:** যদি x + y = 7 এবং xy = 12 হয়, তাহলে x² + y², x³ + y³ এবং (x − y) বের করো।

```
x² + y² = (x+y)² − 2xy = 49 − 24 = 25
(x−y)² = (x+y)² − 4xy = 49 − 48 = 1 → x−y = ±1
x³ + y³ = (x+y)(x²−xy+y²) = 7 × (25−12) = 7 × 13 = 91
```

**প্রশ্ন ২:** যদি a + b + c = 0 হয়, তাহলে a³ + b³ + c³ = ?

```
সূত্র: a + b + c = 0 হলে a³ + b³ + c³ = 3abc
```

**প্রশ্ন ৩:** (99)² = ?

```
(99)² = (100−1)² = 10000 − 200 + 1 = 9801
```

**প্রশ্ন ৪:** (103 × 97) = ?

```
= (100+3)(100−3) = 100² − 3² = 10000 − 9 = 9991
```

## ৩.১৬ দ্বিঘাত সমীকরণ (Quadratic Equations)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সমীকরণ: ax² + bx + c = 0

সূত্র ১:   মূল (Roots): x = [−b ± √(b²−4ac)] / 2a

সূত্র ২:   বৈষম্য (Discriminant): D = b² − 4ac
           D > 0 → দুটো বাস্তব ভিন্ন মূল
           D = 0 → দুটো বাস্তব সমান মূল
           D < 0 → দুটো জটিল মূল

সূত্র ৩:   মূলদ্বয়ের সম্পর্ক (α, β):
           α + β = −b/a   [মূলদ্বয়ের যোগফল]
           α × β = c/a    [মূলদ্বয়ের গুণফল]

সূত্র ৪:   α² + β² = (α+β)² − 2αβ
           α³ + β³ = (α+β)³ − 3αβ(α+β)
           (α − β)² = (α+β)² − 4αβ

সূত্র ৫:   মূলদ্বয় α, β হলে সমীকরণ:
           x² − (α+β)x + αβ = 0
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** x² − 5x + 6 = 0 সমীকরণের মূলদ্বয়ের বর্গের যোগফল কত?

```
α + β = 5, αβ = 6
α² + β² = (α+β)² − 2αβ = 25 − 12 = 13
```

**প্রশ্ন ২:** যে দ্বিঘাত সমীকরণের মূলদ্বয় 3 ও −4, সেটি লেখো।

```
x² − (3 + (−4))x + (3 × (−4)) = 0
x² − (−1)x + (−12) = 0
x² + x − 12 = 0
```

**প্রশ্ন ৩:** 2x² − 7x + 3 = 0 সমীকরণের মূলদ্বয় কত?

```
x = [7 ± √(49−24)] / 4 = [7 ± √25] / 4 = [7 ± 5] / 4
x₁ = 12/4 = 3,  x₂ = 2/4 = 1/2
```

## ৩.২০ ত্রিভুজ (Triangle) — সম্পূর্ণ সূত্র

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সাধারণ সূত্র:
  ক্ষেত্রফল = ½ × ভূমি × উচ্চতা
  তিন কোণের যোগফল = ১৮০°
  বহিঃস্থ কোণ = অন্য দুই অন্তঃস্থ কোণের সমষ্টি

সমবাহু ত্রিভুজ (Equilateral):
  ক্ষেত্রফল = (√3/4) × a²
  উচ্চতা h = (√3/2) × a
  পরিধি = 3a

সমদ্বিবাহু ত্রিভুজ (Isosceles):
  ক্ষেত্রফল = (b/4)√(4a² − b²)
  [a = সমান বাহু, b = ভূমি]

লম্বকোণী ত্রিভুজ (Right-angled):
  পিথাগোরাস: a² + b² = c²  [c = অতিভুজ]
  ক্ষেত্রফল = ½ × ভূমি × লম্ব

হেরোনের সূত্র (যেকোনো ত্রিভুজ):
  s = (a + b + c)/2  [অর্ধপরিধি]
  ক্ষেত্রফল = √[s(s−a)(s−b)(s−c)]

পরিবৃত্তের ব্যাসার্ধ R:
  R = abc / (4 × ক্ষেত্রফল)
  বা R = a / (2sinA)

অন্তর্বৃত্তের ব্যাসার্ধ r:
  r = ক্ষেত্রফল / s
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## ৩.২১ চতুর্ভুজ (Quadrilateral) — সম্পূর্ণ সূত্র

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
চতুর্ভুজের চার কোণের যোগফল = ৩৬০°

আয়তক্ষেত্র (Rectangle):
  ক্ষেত্রফল = দৈর্ঘ্য × প্রস্থ = l × b
  পরিসীমা = 2(l + b)
  কর্ণ = √(l² + b²)

বর্গক্ষেত্র (Square):
  ক্ষেত্রফল = a²
  পরিসীমা = 4a
  কর্ণ = a√2
  কর্ণ দিয়ে ক্ষেত্রফল = d²/2

সামান্তরিক (Parallelogram):
  ক্ষেত্রফল = ভূমি × উচ্চতা = b × h
  পরিসীমা = 2(a + b)

ট্রাপিজিয়াম (Trapezium):
  ক্ষেত্রফল = ½ × (a + b) × h
  [a, b = সমান্তরাল বাহু, h = উচ্চতা]

রম্বস (Rhombus):
  ক্ষেত্রফল = ½ × d₁ × d₂  [d₁, d₂ = কর্ণদ্বয়]
  পরিসীমা = 4a
  বাহু a = ½√(d₁² + d₂²)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** একটি আয়তক্ষেত্রের পরিসীমা ৪৮ সেমি এবং কর্ণ ২০ সেমি। ক্ষেত্রফল কত?

```
2(l + b) = 48 → l + b = 24
l² + b² = 400  (কর্ণ = 20)
(l + b)² = l² + 2lb + b²
576 = 400 + 2lb
2lb = 176 → lb = 88
ক্ষেত্রফল = lb = 88 বর্গ সেমি
```

**প্রশ্ন ২:** একটি রম্বসের কর্ণদ্বয় ১৬ ও ১২ সেমি। ক্ষেত্রফল ও বাহু কত?

## ৩.২২ বৃত্ত (Circle) — সম্পূর্ণ সূত্র

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
মূল সূত্র:
  পরিধি = 2πr = πd
  ক্ষেত্রফল = πr²
  π ≈ 3.1416 ≈ 22/7

চাপের সূত্র:
  চাপের দৈর্ঘ্য = (θ/360°) × 2πr
  খণ্ডক্ষেত্র (Sector) ক্ষেত্রফল = (θ/360°) × πr²

বৃত্তখণ্ড (Segment):
  ক্ষেত্রফল = খণ্ডক ক্ষেত্রফল − ত্রিভুজের ক্ষেত্রফল

বৃত্ত সংক্রান্ত উপপাদ্য:
  কেন্দ্র থেকে জ্যা-র উপর লম্ব জ্যাকে সমদ্বিখণ্ড করে
  একই চাপের উপর কেন্দ্রস্থ কোণ = ২ × পরিধিস্থ কোণ
  অর্ধবৃত্তে কোণ = ৯০°
  দুটি সমান জ্যা কেন্দ্র থেকে সমদূরবর্তী
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

```

```

## ৩.২৩ ঘনবস্তু (3D Solids) — সূত্র

```
  আয়তন = (4/3)πr³

  আয়তন = (2/3)πr³
```

## ৩.২৫ বিন্যাস ও সমাবেশ (Permutation & Combination)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
বিন্যাস (Permutation) — ক্রম গুরুত্বপূর্ণ:
  nPr = n! / (n−r)!
  n! = n × (n−1) × (n−2) × ... × 1
  0! = 1

সমাবেশ (Combination) — ক্রম অগুরুত্বপূর্ণ:
  nCr = n! / [r! × (n−r)!]
  nCr = nC(n−r)
  nC0 = nCn = 1
  nC1 = n
  nCr + nC(r−1) = (n+1)Cr

সম্পর্ক:
  nPr = nCr × r!

বিশেষ বিন্যাস:
  একসাথে বসতে হলে → ব্লক হিসেবে গণনা
  সবসময় আলাদা বসতে হলে → বাকি সাজিয়ে ফাঁকে বসানো
  বৃত্তাকার বিন্যাস = (n−1)!
  গলায় মালা বিন্যাস = (n−1)!/2
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** ৬ জনের মধ্য থেকে ৪ জন নির্বাচন করে সভাপতি, সহ-সভাপতি, সম্পাদক ও কোষাধ্যক্ষ নির্বাচন কত উপায়ে করা যায়?

```
ক্রম গুরুত্বপূর্ণ → বিন্যাস
6P4 = 6!/(6−4)! = 720/2 = 360 উপায়
```

**প্রশ্ন ২:** ১০ জনের মধ্য থেকে ৩ জনের কমিটি কত উপায়ে গঠন করা যায়?

```
ক্রম অগুরুত্বপূর্ণ → সমাবেশ
10C3 = 10!/(3! × 7!) = (10×9×8)/(3×2×1) = 720/6 = 120 উপায়
```

**প্রশ্ন ৩:** "MATHEMATICS" শব্দের অক্ষরগুলো দিয়ে কত ধরনের বিন্যাস করা যায়?

```
MATHEMATICS: M(2), A(2), T(2), H, E, I, C, S = 11 অক্ষর
বিন্যাস = 11! / (2! × 2! × 2!) = 39,916,800/8 = 4,989,600
```

**প্রশ্ন ৪:** ৬ জন একটি গোল টেবিলে কত উপায়ে বসতে পারে?

```
বৃত্তাকার বিন্যাস = (6−1)! = 5! = 120 উপায়
```

## ৩.২৬ সম্ভাবনা (Probability)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   P(A) = n(A) / n(S)
            n(A) = অনুকূল ঘটনার সংখ্যা
            n(S) = মোট ঘটনার সংখ্যা

সূত্র ২:   0 ≤ P(A) ≤ 1
            P(A) + P(A') = 1
            P(নিশ্চিত) = 1
            P(অসম্ভব) = 0

সূত্র ৩:   P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
            পারস্পরিক বিচ্ছিন্ন: P(A ∪ B) = P(A) + P(B)

সূত্র ৪:   স্বাধীন ঘটনা:
            P(A ∩ B) = P(A) × P(B)

সূত্র ৫:   শর্তসাপেক্ষ সম্ভাবনা:
            P(A|B) = P(A ∩ B) / P(B)

বিশেষ ঘটনা:
  একটি মুদ্রায় Head বা Tail: P = 1/2
  একটি ছক্কায় যেকোনো সংখ্যা: P = 1/6
  তাস থেকে (52 তাস):
    একটি এক্কা: 4/52 = 1/13
    একটি লাল তাস: 26/52 = 1/2
    একটি King: 4/52 = 1/13
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** একটি ব্যাগে ৪টি লাল ও ৬টি সাদা বল আছে। এলোমেলোভাবে ২টি বল তুললে উভয় লাল হওয়ার সম্ভাবনা কত?

```
P(উভয় লাল) = 4C2 / 10C2
             = 6 / 45 = 2/15
```

**প্রশ্ন ২:** দুটি ছক্কা একসাথে নিক্ষেপ করলে যোগফল ৮ হওয়ার সম্ভাবনা কত?

```
মোট ঘটনা = 6 × 6 = 36
যোগফল ৮ হওয়ার উপায়: (2,6),(3,5),(4,4),(5,3),(6,2) = 5টি
P = 5/36
```

**প্রশ্ন ৩:** A ও B স্বাধীনভাবে একটি সমস্যা সমাধান করতে পারে। A-এর সম্ভাবনা 2/3 এবং B-এর 3/4। অন্তত একজন সমাধান করতে পারবে তার সম্ভাবনা কত?

```
P(A না) = 1 − 2/3 = 1/3
P(B না) = 1 − 3/4 = 1/4
P(কেউই না) = 1/3 × 1/4 = 1/12
P(অন্তত একজন) = 1 − 1/12 = 11/12
```

## ৩.২৭ সেট তত্ত্ব (Set Theory)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
সূত্র ১:   n(A ∪ B) = n(A) + n(B) − n(A ∩ B)

সূত্র ২:   n(A ∪ B ∪ C) = n(A) + n(B) + n(C)
                          − n(A∩B) − n(B∩C) − n(A∩C)
                          + n(A∩B∩C)

সূত্র ৩:   কেবল A-তে আছে = n(A) − n(A∩B)
            কেবল B-তে আছে = n(B) − n(A∩B)
            কোনোটিতেই নেই = n(U) − n(A∪B)

ডি মর্গানের সূত্র:
  (A ∪ B)' = A' ∩ B'
  (A ∩ B)' = A' ∪ B'
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** ১০০ জন শিক্ষার্থীর মধ্যে ৬০ জন গণিত, ৫০ জন বিজ্ঞান পড়ে। উভয় পড়ে ৩০ জন। শুধু গণিত বা শুধু বিজ্ঞান পড়ে কতজন?

```
n(M ∪ S) = 60 + 50 − 30 = 80
শুধু গণিত = 60 − 30 = 30
শুধু বিজ্ঞান = 50 − 30 = 20
কোনোটিই না = 100 − 80 = 20
```

**প্রশ্ন ২:** ১৫০ জনের একটি দলে, ৯০ জন চা পান করে, ৭০ জন কফি পান করে, ৩০ জন উভয়ই পান করে। কেউ পান করে না কতজন?

```
n(T ∪ C) = 90 + 70 − 30 = 130
কেউ পান করে না = 150 − 130 = 20 জন
```

## ৩.২৮ পরিসংখ্যান (Statistics)

### মূল সূত্রসমূহ

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
পরিমাপের কেন্দ্রীয় প্রবণতা:

গাণিতিক গড় (Mean):
  x̄ = Σx / n   (অবিন্যস্ত তথ্য)
  x̄ = Σfx / Σf  (বিন্যস্ত তথ্য)

মধ্যমা (Median):
  n = বিজোড়: মধ্যমা = [(n+1)/2]-তম পদ
  n = জোড়: মধ্যমা = [(n/2)-তম + (n/2+1)-তম] / 2

সংখ্যাগুরু (Mode):
  সবচেয়ে বেশিবার আসা মান

সম্পর্ক:
  Mode = 3 × Median − 2 × Mean  (বিকৃত বণ্টনে)

বিস্তার (Range) = সর্বোচ্চ − সর্বনিম্ন

পার্থক্যমান (Variance):
  σ² = Σ(x − x̄)² / n

আদর্শ বিচ্যুতি (Standard Deviation):
  σ = √[Σ(x − x̄)² / n]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### বিস্তারিত উদাহরণ

**প্রশ্ন ১:** তথ্য: 4, 7, 8, 3, 9, 5, 7, 7, 6, 4। গড়, মধ্যমা ও সংখ্যাগুরু বের করো।

